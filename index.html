デバイス別・確実にダウンロードする方法
✅ PC ✅ iPhone ✅ Android ⚠ 署名動画は外部ツール推奨
💻 PC（Windows / Mac）
✅ 最も確実な方法は yt-dlp を使うことです。
コマンドラインで動作し、限定公開・署名動画・高画質・音声付きすべてに対応します。
1
yt-dlp をインストール
winget install yt-dlp.yt-dlp（Windows）
brew install yt-dlp（Mac）
2
FFmpeg もインストール（音声と動画の結合に必要）
winget install Gyan.FFmpeg（Windows）
brew install ffmpeg（Mac）
3
コマンドを実行
yt-dlp -f "bestvideo+bestaudio" --merge-output-format mp4 "YouTubeのURL"

📱 iPhone（Safari）
✅ iPhoneで確実なのは Cobalt.tools を使う方法です。
アプリ不要・ブラウザだけで完結します。
1
Safariで cobalt.tools を開く
2
YouTubeの動画URLをコピーして貼り付け
3
矢印ボタンを押す → ダウンロード開始

📌 Failed to fetch エラーについて
このエラーは CORS（クロスオリジン）制限 が原因です。
ブラウザは YouTube のサーバーから直接データを取得することを許可していません。
そのため、外部サービス（上記リンク） を使うのが確実な解決策です。
🚨 共通の注意事項
  • ブラウザ単体では署名（cipher）が必要な動画はダウンロードできません
  • 年齢制限・メンバー限定・ライブ配信中の動画は対象外です
  • iPhone Safari では download 属性が無視される場合があります
  • 確実性を求めるなら yt-dlp（PC） が最強です
  • YouTubeの利用規約を遵守し、動画の権利者を尊重してください
