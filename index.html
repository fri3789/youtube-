📱 iPhone（Safari）
✅ iPhoneで確実な方法は Cobalt.tools を使うことです。
アプリ不要・ブラウザだけで完結・広告なし・高画質。限定公開動画にも対応しています。
🔥 推奨サービス
⚡ Cobalt.tools 🏆 最強おすすめ！広告なし・最速・高画質・全デバイス対応 📥 SaveFrom.net 老舗・安定・多機能（広告あり） 💾 SaveTube.app ウォーターマークなし・シンプル
📖 使い方（たった3ステップ）
1
SafariでYouTubeを開き、ダウンロードしたい動画の URLをコピー
2
上の推奨サービス（Cobalt.tools など）をタップして開く
3
URLを貼り付けて ダウンロード をタップ → 保存完了！
📸 写真フォルダに保存する方法
ダウンロード後、「ファイル」アプリを開く → ダウンロードした動画をタップ → 共有ボタン → 「ビデオを保存」 を選ぶと写真フォルダに移せます。
⚠ iPhoneでの注意点
  • YouTubeアプリではなく SafariでYouTubeを開く 必要があります
  • 年齢制限・メンバー限定動画はダウンロードできません
  • Cobalt.tools が最も安定しています

方法A：ブラウザで完結（おすすめ）
⚡ Cobalt.tools 🏆 最強おすすめ！広告なし・最速・高画質 📥 SaveFrom.net 老舗・安定・多機能（広告あり） 📥 Loader.to 高画質対応・安定（広告あり）
1
YouTube動画の URLをコピー
2
上のサービスを 開いてURLを貼り付け
3
ダウンロード をクリック → 完了！
方法B：yt-dlp（100%確実・限定公開・署名動画もOK）
1
yt-dlp と FFmpeg をインストール
winget install yt-dlp.yt-dlp（Windows）
brew install yt-dlp ffmpeg（Mac）
2
コマンドを実行
yt-dlp -f "bestvideo+bestaudio" --merge-output-format mp4 "YouTubeのURL"
3
完了！ 動画が保存されます
⚠ PCでの注意点
  • Cobalt.tools が最も手軽で失敗が少ないです
  • yt-dlp は すべての動画（限定公開・署名動画・4K）に対応します
  • 年齢制限・メンバー限定動画は --cookies-from-browser chrome オプションが必要です
